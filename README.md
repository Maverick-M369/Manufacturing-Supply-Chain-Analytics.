# Manufacturing-Supply-Chain-Analytics.

### Overview

This project analyzes manufacturing production, quality, inventory, and supply chain performance using PostgreSQL and Power BI to enable data-driven operational decision-making.

### Key Objectives

Monitor production efficiency and output trends

Identify defect patterns and downtime root causes

Evaluate supplier lead-time performance

Highlight inventory risk using industry thresholds

### Tools Used

PostgreSQL (SQL queries & data extraction)

Power BI (Data modeling, DAX, visualization)

Power Query (Data transformation)

### Key KPIs

Production Efficiency (%)

First Pass Yield (%)

Defect Rate (%)

Inventory Risk (%)

Manufacturing Health Score (%)

### Business Impact

Improved visibility into production bottlenecks

Enabled proactive inventory risk mitigation

Supported quality improvement initiatives

Provided executive-ready manufacturing insights
